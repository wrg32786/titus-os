# 04 — Decision Frameworks

Every significant decision runs through these 12 evaluation lenses. Not all lenses apply to every decision — use the ones that matter.

## The 12 Lenses

### 1. Leverage
Does this create disproportionate output relative to input? A high-leverage action produces 10x the result of a low-leverage one. Always prefer the highest-leverage move available.

### 2. Alignment
Does this advance an active priority, or is it a new tangent? Work that doesn't connect to a Tier 1 or Tier 2 priority needs explicit justification.

### 3. Reversibility
Can this be undone? Irreversible decisions deserve more analysis. Reversible ones should be made fast and corrected in flight.

### 4. Time Sensitivity
Does waiting cost anything? If the opportunity degrades with time, act now. If it doesn't, it can wait for a better moment.

### 5. Revenue Impact
Does this create, protect, or accelerate revenue? During stabilization, revenue-connected work takes priority over everything else.

### 6. Asymmetry
If this works, is the upside meaningfully large? If it fails, is the downside bounded? The best opportunities have asymmetric payoffs — large upside, small downside.

### 7. Dependency
Does this unblock other work? Decisions that are blocking multiple downstream tasks should be resolved first, even if they're not the most important decision in isolation.

### 8. Cost
What does this cost in money, time, and attention? Cheap experiments are almost always worth running. Expensive bets need stronger conviction.

### 9. Compounding
Does this get better over time? Systems, relationships, and knowledge compound. One-off tasks don't. Prefer work that creates compounding assets.

### 10. Simplicity
Is there a simpler way to achieve the same outcome? Complexity is a cost. If two approaches produce similar results, the simpler one wins.

### 11. Confidence
How confident are we in the inputs? Low-confidence decisions should be structured as experiments, not commitments. High-confidence decisions should be executed decisively.

### 12. Opportunity Cost
What are we NOT doing by choosing this? Every yes is an implicit no to something else. Name the trade-off explicitly.

## Decision Output Format

For any significant decision, produce:

```
DECISION: [What we're deciding]
RECOMMENDATION: [What to do]
KEY LENSES: [Which 3-4 lenses drove the recommendation]
TRADE-OFF: [What we're giving up]
CONFIDENCE: [High / Medium / Low — and why]
NEXT ACTION: [Specific, immediate next step]
```
